## Dream Travel Viagens ✈️
# Curadoria de viagens de luxo e experiências sob medida.

# 📖 Sobre o Projeto
A Dream Travel é uma plataforma tecnológica de alto padrão, desenhada para orquestrar viagens exclusivas. O nosso sistema permite que a curadoria (Jackeline) gerencie destinos, conteúdos editoriais (Journal) e, principalmente, disponibilize roteiros personalizados e seguros para clientes VIP através de uma Área Reservada.

# 🚀 Funcionalidades Principais
- Catálogo de Destinos: Exibição dinâmica de locais exclusivos com design editorial.
- Diario de Viagens (Blog): Espaço para relatos de viagem, com estética de revista de luxo.
- Área VIP: Portal restrito onde o cliente tem acesso ao seu roteiro confidencial e suporte direto.
- Control Room (Admin): Painel de administração para gestão total de conteúdo, imagens e acessos de clientes.
- Assistente de Consultoria: Integração estratégica com WhatsApp para atendimento imediato e personalizado.

# 🛠 Tecnologias Utilizadas
- Frontend: React, TypeScript, Tailwind CSS, Framer Motion (para animações fluidas).
- Router: Wouter (navegação leve e performática).
- Gestão de Estado: React Hooks.
- Componentização: UI components baseados em padrões de alta performance.
- Backend (Mock): Sistema de gestão de estados pronto para integração com API RESTful.
